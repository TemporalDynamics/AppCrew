# Visión del piloto — Plataforma de sourcing asistido

> Documento para Rodrigo — versión simplificada del roadmap.

---

## 1. Qué vas a poder probar ahora

- Definir criterios de búsqueda (rol, industria, ubicación, seniority).
- El sistema busca candidatos en fuentes permitidas.
- Los candidatos se guardan en una base reutilizable (sin duplicados entre búsquedas).
- Podés revisar perfiles con vista anonimizada primero (sin nombre, sin foto).
- Aprobás, descartás o pedís más información.
- El sistema registra evidencia de por qué aparece cada candidato.
- Dashboard web para ver resultados y tomar decisiones.

## 2. Qué límites tiene el piloto

- Es una cuenta privada dentro de un entorno controlado.
- Volumen inicial acotado (técnico, operativo y presupuestario).
- Fuentes disponibles: información que vos cargues, APIs autorizadas y datos públicos accesibles.
- No hay acceso a perfiles privados, cuentas cerradas ni scraping no autorizado.
- No hay contacto automático con candidatos sin revisión humana.
- No hay integración con ATS todavía (puede crecer después).

## 3. Qué decisiones quedan en manos humanas

- Todo: selección, contacto, entrevista, descarte, contratación.
- La plataforma sugiere, ordena y muestra evidencia, pero no decide.
- La revisión ciega (sin nombre ni datos identificatorios al principio) te permite evaluar encaje antes de saber quién es.
- Podés revelar identidad cuando quieras.
- El sistema registra cada decisión y por qué se tomó.

## 4. Qué fuentes se pueden usar

- Datos que vos cargues (CVs, bases propias que tengas derecho a usar).
- APIs autorizadas (Torre.co, Firecrawl, Brave Search).
- Información pública dentro de límites razonables.
- No se usan cuentas personales, sesiones privadas ni credenciales de terceros.
- Cada candidato tiene registro de: fuente, fecha, evidencia y nivel de confianza.

## 5. Qué puede crecer después

- Más fuentes y conectores.
- Búsquedas de mayor volumen (cohortes, lotes, coverage reports).
- Remote operator mode (el agente con ojos y manos sobre entorno controlado).
- Integración con ATS y exportación de datos.
- Modo 24/7 en VPS (residencia operativa).
- Si el piloto funciona, se conversa el siguiente paso.

---

**La plataforma no es solo una web de recruiting. Es un sistema operativo administrado de agentes de sourcing, con memoria viva de talento, revisión humana, fuentes autorizadas y capacidad futura de operación remota supervisada.**
